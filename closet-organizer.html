To enable Google login and cloud sync, you need to create a free Firebase project (takes ~5 minutes).

Step 1: Create Firebase Project

  1. Go to Firebase Console
  2. Click Create a project
  3. Name it "Closet Organizer" (or anything)
  4. Disable Google Analytics (optional) ‚Üí Create

Step 2: Enable Authentication

  1. In Firebase Console, go to Build ‚Üí Authentication
  2. Click Get started
  3. Click Google under Sign-in providers
  4. Toggle Enable, add your email as support email
  5. Click Save

Step 3: Create Firestore Database

  1. Go to Build ‚Üí Firestore Database
  2. Click Create database
  3. Select Start in test mode (for now)
  4. Choose a location ‚Üí Click Enable

Step 4: Add Web App & Get Config

  1. Go to Project settings (gear icon)
  2. Scroll to "Your apps" ‚Üí Click web icon </>
  3. Name it "Closet Organizer" ‚Üí Register
  4. Copy the firebaseConfig object
